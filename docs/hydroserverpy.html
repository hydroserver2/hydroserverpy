Getting StartedÔÉÅ

HydroServer Python ClientÔÉÅ

The hydroserverpy Python package provides an interface for managing HydroServer data and metadata, loading observations, and performing data quality control. This guide will go over how to install the package and connect to a HydroServer instance.

InstallationÔÉÅ

You can install the package via pip:

pip install hydroserverpy

Connecting to HydroServerÔÉÅ

To connect to HydroServer, you need to initialize the client with the instance of HydroServer you‚Äôre using and your user credentials if you want to access and modify your own data. If you don‚Äôt provide authentication credentials you can read public data, but you will not be able to create or modify any data.

Example: Anonymous UserÔÉÅ
from hydroserverpy import HydroServer

# Initialize HydroServer connection.
hs_api = HydroServer(
    host='https://playground.hydroserver.org'
)
Example: Basic AuthenticationÔÉÅ
from hydroserverpy import HydroServer

# Initialize HydroServer connection with credentials.
hs_api = HydroServer(
    host='https://playground.hydroserver.org',
    username='user@example.com',
    password='******'
)
